# Xeno Transaction Intelligence Platform

A Streamlit-based platform for validating transaction datasets, identifying data quality issues, generating business insights, and exporting professional PDF reports.

## Features

- Data Validation
- Quality Score Assessment
- Executive Summary
- Analytics Dashboard
- Validation Exceptions
- PDF Report Generation
- CSV Export
- Dataset Chunking

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- ReportLab

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
